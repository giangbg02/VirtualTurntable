    Select Album Directory

     

    Manual

    Please know this program is meant to be used locally at this time. Web implementation to be added in the future. Download here.

    Please know this program requires Google Chrome or Firefox v50.0.2 to run correctly.

    1. Select an album directory by clicking browse. (Note: Directory must be placed in website directory)

    2. Wait a few moments and list will populate. (Note: List may show undefined. If so, click Refresh Tags to correct)

    3. Once list has populated, click on track to play song.

    4. To change album directory, click Reset button to refresh page. (Note: Changing directory more than once without reset will cause issues)

    Warning: Using non-MP3 files will result in issues at this time

    If having trouble viewing, zoom out browser level

    Features:

    1. Changeable background (Color options in bottom right corner)

    2. Spinning Disc (Click on album art)
